# Web Portfolio

This is my web portfolio where i show my projects.

## Description

This web portfolio is built with [Astro](https://astro.build) v6 and
[Tailwind CSS](https://tailwindcss.com) v4. The UI is split into Astro
components under `src/components/`, the original design lives in
`src/styles/global.css`, and static assets are served from `public/`.

## Getting Started

### Prerequisites

* Node.js 18+ and npm

### Install dependencies

```bash
npm install
```

### Available commands

| Command           | Action                                       |
| ----------------- | -------------------------------------------- |
| `npm run dev`     | Start the dev server at `localhost:4321`     |
| `npm run build`   | Build the production site to `./dist/`       |
| `npm run preview` | Preview the production build locally         |
